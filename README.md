## iOS Apps

### iOS Communication

- [**Signal**](https://github.com/signalapp/Signal-iOS) - Free, world-wide, private messaging and phone calls app with end-to-end encryption. Features include disappearing messages, screen security, and relay calls to protect your IP address.

- [**Telegram**](https://github.com/TelegramMessenger/Telegram-iOS) - Messaging app focused on speed and security with cloud-based storage and sync. Supports channels, groups, bots, and file sharing with no size limits.

- [**Element**](https://github.com/element-hq/element-ios) - Secure decentralised chat and VoIP app built on the Matrix protocol. Offers end-to-end encryption, bridges to other chat networks, and self-hosting options.

- [**ChatSecure**](https://github.com/ChatSecure/ChatSecure-iOS) - Free and open source encrypted chat app using OTR encryption over XMPP. Supports multiple accounts, offline messaging, and Tor for anonymity.

- [**Monal**](https://github.com/monal-im/Monal) - Modern XMPP chat client that doesn't require giving third parties access to passwords or messages. Features OMEMO encryption, push notifications, and file transfers.

- [**Wire**](https://github.com/wireapp/wire-ios) - Modern, private communications with crystal clear voice and video calls. Provides end-to-end encryption for all conversations and supports team collaboration features.

- [**Jami**](https://review.jami.net/admin/repos/ring-client-ios) - Free and universal communication platform preserving privacy and freedoms. Offers distributed peer-to-peer architecture with no central server or authority.

- [**Threema**](https://github.com/threema-ch/threema-ios) - Secure, anonymous, and private messenger app with no phone number required. Generates a unique Threema ID and stores contacts locally on your device.

- [**deltachat**](https://github.com/deltachat/deltachat-ios) - Email-based instant messaging that works with your existing email provider. Combines the reliability of email with the convenience of messenger apps.

- [**ProtonMail**](https://github.com/ProtonMail/ios-mail) - Encrypted email app providing end-to-end and zero-access encryption. Features self-destructing messages, password-protected emails, and Swiss privacy laws protection.
